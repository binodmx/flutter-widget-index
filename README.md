# Flutter Widget Index

Mobile app to explore flutter widgets.

## Usages

- ✅ List all flutter widgets.
- ✅ View widget details.
- ✅ Search widgets by name.
- ❌ Filter widgets by tags.
- ❌ Mark widgets as favorite.
